## Populating the `<head>` of your document

### Add tags in `root.tsx`

### Use `solid-meta` inside component tree

- Fill up head in `root.tsx` (e.g., favicon)
  - Links, Meta, Scripts components (people won't be removing these)
  - Meta
    - solid-meta on the server (e.g., title tag)
    - populate `head` section dynamically with `solid-meta`
  - Links
    - prefetch CSS in head on per-route basis (prod only)
  - Scripts
    - enables JS (links to entry client file). Removing it disables JS in your app.
